🕹️ Modern Memory Match
A high-performance, responsive memory game built with Vanilla JavaScript, CSS3, and HTML5. This project features dynamic difficulty scaling, 3D card-flip animations, and move-limited gameplay.

🚀 Features
Dynamic Grid Logic: Automatically generates 3X4, 4X6, 6X8 grids based on difficulty.
Move-Limited Challenges: Each mode has a maximum move threshold. Win before you run out of turns!
3D Animations: Smooth card flipping using CSS perspective and preserve-3d.
Audio Integration: Instant feedback with click, win, and lose sound effects.
Responsive UI: Mobile-friendly design using CSS Grid and Flexbox.

🛠️ Tech Stack
HTML5: Semantic structure and DOM elements.

CSS3: Custom properties (variables), Grid layout, and 3D transforms.

JavaScript: ES6+ logic, event-driven programming, and state management.
