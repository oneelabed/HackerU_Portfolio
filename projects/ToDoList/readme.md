📝 Task Manager Pro
A feature-rich, high-performance To-Do List application built with pure Vanilla JavaScript, HTML5, and CSS3. This project demonstrates core frontend engineering concepts including State Management, DOM Manipulation, Data Persistence, and CRUD operations.

🚀 Key Features
Full CRUD Functionality: Create, Read, Update (Inline), and Delete tasks seamlessly.

Data Persistence: Uses localStorage to ensure your tasks and theme preferences survive page refreshes.

Advanced Filtering: Sort tasks by status (All, Active, Completed) or find specific items using the real-time Search Bar.

Priority & Deadlines: Assign "Low", "Medium", or "High" priority levels and set due dates for better time management.

Interactive Dashboard: Real-time progress bar and task counters provide immediate feedback on productivity.

Dark Mode: A fully implemented dark theme with persistent settings.

Drag & Drop: Native JavaScript Drag and Drop API integration for manual task reordering.

Refined UX: Includes micro-animations, custom-styled checkboxes, and "Empty State" handling for a polished feel.

🛠️ Technologies Used
HTML5: Semantic structure and Native Drag & Drop API.

CSS3: Custom variables (CSS Properties), Flexbox layout, and Keyframe animations.

JavaScript (ES6+): Vanilla JS for logic, LocalStorage for state, and event-driven architecture.

Icons: FontAwesome 6.0.
