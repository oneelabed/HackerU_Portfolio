⚪ Tic-Tac-Toe
A minimalist, high-performance Tic-Tac-Toe game built with vanilla JavaScript. This version features a "Middle-Ground" UI designed specifically to reduce eye strain—avoiding the harshness of pure white and the darkness of deep black.

🚀 Features
Nordic Slate Aesthetic: A custom-designed UI using muted blue-grays and soft shadows for a comfortable gaming experience.

Live Scoreboard: Tracks wins for Player X and Player O in real-time.

Smart Rendering: Utilizes asynchronous JavaScript (setTimeout) to ensure game markers (X/O) appear on the screen before win alerts trigger.

Responsive Logic: Built-in tie detection and automatic board clearing.

Technology              Use Case
HTML5                   Semantic structure and game grid
CSS3                    Flexbox/Grid layout & "Nordic" styling
JavaScript              Game logic, DOM manipulation, & Event Loop management
