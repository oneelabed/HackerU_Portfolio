🧩 Sudoku
A clean, modern, and fully functional Sudoku game built using only HTML5, CSS3, and Vanilla JavaScript. No frameworks or external libraries required.

🚀 Features
Difficulty Selection: Choose between Easy, Medium, and Hard modes.

Curated Puzzles: Includes 10 hand-picked, mathematically proven solvable puzzles (including the famous AI Escargot).

Dynamic Countdown: Difficulty-based time limits that increase the challenge:

Easy: 10 Minutes

Medium: 15 Minutes

Hard: 20 Minutes

Smart Grid Design: Classic Sudoku layout with bold 3x3 section borders and a "focus" state for the active cell.

Validation Engine: A built-in logic checker that verifies rows, columns, and 3x3 subgrids.

Game Over State: Automatically locks the board when the timer hits zero to prevent further moves.
