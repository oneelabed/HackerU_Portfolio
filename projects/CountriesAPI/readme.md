🌍 World Explorer
A dynamic, responsive web application that allows users to explore data from over 250 countries using the Rest Countries API. This project features real-time searching, region filtering, and detailed modal views.

✨ Features
Real-time Search: Instantly filter countries by name as you type.
Region Filtering: Narrow down your exploration by continent (Africa, Americas, Asia, Europe, Oceania).
Detailed Modals: Click any country card to see expanded information, including population (formatted), capital, and currencies.
External Mapping: Direct links to view the country's location on Google Maps.
Optimized Performance: * Pagination: Loads 12 countries at a time to keep the UI snappy.
Lazy Loading: Images only download as they enter the viewport.
API Efficiency: Fetches only the specific fields required.
Responsive Design: Fully functional on mobile, tablet, and desktop views using CSS Grid and Flexbox.

🛠️ Tech Stack
HTML5: Semantic structure.
CSS3: Custom styling with Grid, Flexbox, and Glassmorphism effects.
JavaScript (ES6+): Fetch API for asynchronous data.
Intersection Observer / Pagination logic.
Dynamic DOM manipulation.
